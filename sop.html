SOP LAYANAN JOKI CLASH OF CLANS

Standar Operasional Prosedur untuk Keamanan dan Kelancaran Akun

🚀 BAGIAN 1: PROSEDUR UMUM

  • Pembersihan Sesi: Client wajib Force Close aplikasi CoC dan tidak membukanya hingga instruksi "Selesai".
  • Jeda Keamanan (Cooldown): Joki akan login minimal 2 jam setelah konfirmasi Log Out untuk menyamarkan IP.
  • Dokumentasi: Joki mengirimkan screenshot profil & resource awal sebagai bukti validitas data.

Skenario A: Proteksi Aktif

  • Client wajib standby untuk memberikan Kode SMS/Email saat login awal.
  • Joki dilarang log out/hapus data agar tidak perlu minta kode berulang kali.
  • Jika Account Locked, Client wajib memberikan Recovery Code.
  • Risiko: Akun hangus jika Recovery Code hilang/lupa.

Skenario B: Proteksi Nonaktif

  • Client memberikan kode verifikasi Supercell ID via email.
  • Joki dilarang keras mengaktifkan Proteksi Akun (PA) dengan nomor pribadi.
  • Pasca-Joki: Client wajib segera mengaktifkan kembali PA dan ganti password email.
  • Saran: Segera aktifkan PA setelah serah terima kembali.

Perbandingan Risiko

Fitur PA AKTIF PA NONAKTIF
Keamanan Sangat Tinggi Rendah
Kelancaran Terhambat (Butuh Koordinasi) Sangat Lancar
